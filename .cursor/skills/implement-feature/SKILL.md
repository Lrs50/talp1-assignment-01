# Skill: implement-feature

## Goal
Implement a feature in a clean and readable way

## Steps
1. Implement backend first (controller → service → repository)
2. Then implement frontend
3. Keep functions small and focused

## Rules
- Follow AGENTS.md strictly
- Use clear naming
- Avoid complex logic in a single function
- Do not overengineer

## Output
- Code with clear structure
- Brief explanation
